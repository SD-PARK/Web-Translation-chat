아래 링크를 통해 앱을 테스트해볼 수 있습니다.

## 주요 기능

### 계정 관련

- 계정 생성, 삭제
    - 계정 중복 검사
- 로그인, 로그아웃
    - 로그인 정보 일치 여부 확인
    - 비밀번호 암호화(SHA256)
- 마이페이지

### 커뮤니케이션

- 친구 및 채팅방 목록 출력
- 친구 추가, 삭제
- 새 채팅방 개설, 참가
- 채팅방 초대, 나가기
    - 채팅방 초대 코드 부여
- 채팅 내역 출력
- 실시간 대화
- 자동 번역

### 언어

- 언어 설정(설정 국가에 따른 페이지 내 텍스트 변경)
